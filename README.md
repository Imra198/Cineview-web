# Cineview-web
A responsive movie and TV show discovery website built with HTML and Tailwind CSS. Includes sections for Hollywood news, trending movies, and TV shows.

Features

✔️ Responsive layout using Tailwind CSS
✔️ Sticky navigation bar with smooth styling
✔️ Dedicated pages:
  • Home
  • Hollywood
  • Movies
  • TV Shows
✔️ Trending movies grid

Project Structure
cineview-web/
│── index.html
│── hollywood.html
│── movies.html
│── tv-shows.html
│── /assets
│    └── images/
│── README.md

Technologies Used
HTML5
Tailwind CSS (CDN version)
Google Images (Temporary placeholders)

How to Use
Open index.html in your browser
Click navigation buttons to switch between sections
Explore trending movies and TV shows
